# DecodeLabs Python Industrial Training

This repository holds my code and milestones from the DecodeLabs Python Industrial Training program. Everything here focuses on building solid backend fundamentals, managing data cleanly, and making sure programs don't crash when users type the wrong thing.

---

## What I'm Learning & Practicing

*   **Handling App State:** Learning how to keep track of changing data in real-time and understanding where to declare variables so data doesn't accidentally reset mid-loop[cite: 1].
*   **Error Proofing (Defensive Coding):** Using `try-except` blocks to catch user input mistakes early so the script keeps running smoothly instead of crashing out[cite: 1].
*   **Core Logic:** Getting comfortable with loops, conditional logic, and basic data structures to handle continuous data entry.
*   **Clean Git Workflow:** Using Git and GitHub properly to keep track of updates, write clear commit messages, and manage a clean project directory.
